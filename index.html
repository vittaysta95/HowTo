    Sync across devices

    This app has no server of its own. It uses a private GitHub Gist as a tiny free database so your progress can follow you between phone and laptop. Create a fine-grained personal access token scoped only to "Gists: Read and write" at github.com/settings/tokens, and paste it below. It's stored only in this browser's local storage — never sent anywhere except api.github.com.
